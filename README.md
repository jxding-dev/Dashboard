# ⚙️ MY MANAGER

작업, 계획, 프롬프트, 수익 구조까지 한 번에 관리할 수 있는 개인용 통합 관리 웹 애플리케이션입니다.

단순 기록이 아니라
**실행 → 관리 → 정리 → 반복** 흐름을 하나로 묶는 것을 목표로 합니다.

---

## 🔧 주요 기능

### 1. 대시보드

* 체크리스트 진행률 표시
* 예상 월 순수익 자동 계산
* 로드맵 / 프롬프트 상태 요약
* 카드 드래그 정렬 지원 

---

### 2. 체크리스트

* 그룹 단위 관리
* 완료 상태 체크
* 태그(긴급 / 이번주 / 나중에) 분류 

---

### 3. 로드맵

* 월별 목표 및 작업 관리
* 작업 추가 / 삭제 / 수정
* Enter 입력으로 빠른 추가 지원 

---

### 4. 프롬프트 관리

* 카테고리별 분류 (ChatGPT, Claude 등)
* 검색 / 필터 기능
* 복사 / 수정 / 삭제 기능 

---

### 5. 워크플로우 관리

* 작업 단계 기록
* 수정 / 삭제 / 추가 관리
* 작업 흐름 시각화

---

### 6. 플랫폼 관리

* 플랫폼별 전략 기록
* 상태 관리 (활성 / 준비 / 보류)
* 단가 및 전략 정리 

---

### 7. 수익 계산기

* 단가 / 작업 수 / 비용 / 세율 기반 계산
* 예상 순수익 자동 계산
* 결과 저장 기능 

---

* Node.js (정적 서버)

## ⚠️ 참고 사항

* 데이터는 브라우저에 저장되므로 삭제 시 복구 불가
* JSON 백업 / 복원 기능 제공 
* 서버는 단순 정적 파일 제공 용도

---

## 📌 핵심 특징

* 기능 중심이 아닌 **흐름 중심 설계**
* 입력 → 관리 → 결과 확인 → 반복 구조
* 모든 기능이 하나의 상태(state)로 연결됨
